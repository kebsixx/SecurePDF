# SecurePDF Pro

SecurePDF Pro adalah aplikasi web sederhana, aman, dan sepenuhnya offline untuk mengonversi gambar menjadi file PDF berkualitas tinggi langsung di browser Anda. Tidak ada data yang diunggah ke server, menjaga privasi dokumen Anda 100%.

## âœ¨ Fitur Utama

- **Privasi Total**: Pemrosesan dilakukan 100% di sisi klien (browser). Data Anda tidak pernah meninggalkan perangkat Anda.
- **Drag & Drop**: Tarik dan lepas gambar untuk mengunggah dengan mudah.
- **Atur Urutan**: Geser dan atur ulang urutan halaman sesuai keinginan.
- **Orientasi Pintar**: Mendukung Portrait, Landscape, atau **Auto** (menyesuaikan orientasi tiap halaman berdasarkan bentuk gambar).
- **Kompresi Cerdas**: Pilih kualitas gambar (Rendah, Sedang, Tinggi) untuk mengontrol ukuran file PDF hasil akhir.
- **Preview PDF**: Lihat hasil PDF sebelum mengunduhnya.
- **Kustomisasi**: Atur ukuran kertas (A3, A4, A5, Letter, Legal), margin, dan nama file kustom.
